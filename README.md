# Tic-Tac-Toe Game

A modern implementation of the classic Tic-Tac-Toe game using HTML, CSS, and JavaScript.

## Features
- **Clean, Modern UI** - Warm color palette with smooth animations and transitions
- **Score Tracking** - Scoreboard tracking wins for both players
- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Winner Animations** - Celebration effects highlighting the winning combinations
- **Winner Popup** - Elegant winner announcements with button to play again


## Game Rules
- Two players take turns placing X and O marks on a 3Ã—3 grid
- Player X always goes first
- The first player to get three marks in a row (horizontally, vertically, or diagonally) wins
- If all 9 cells are filled with no winner, the game ends in a draw
- Scores persist across multiple rounds
